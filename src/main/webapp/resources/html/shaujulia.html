Julia Set Page

   

Change Zoom Level of Julia Set

I've always loved fractals and this Julia Set especially so. Gaston Maurice Julia realised the mathematics for this at the beginning of the 20th Century and it's only with the advent of computer rendering that we have been able to appreciate their beauty. This simple implementation uses Javascript to render the set on an HTML Canvas and I think it's pretty quick considering what it's doing. It's not even utilising WebGL.
